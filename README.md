# LOL.ai

This website is videcoded using google Antigravity and it is deployed through Vercel.

To replicate this:

1. Clone or fork this repo as a new github repository
2. Naviagte to Vercel, import your github repository
3. The webstie used the designer's API from GNEWs, Arxiv, Semantic Scholar and OpenAlex, all APIs are free tiers which means they WILL hit rate limit (100 daily request for GNews). If you need more access or have different APIs, you can change the API on API.js.
4. Leave all the setting as defaulted, change the product name if necessary
5. Deploy on Vercel, and the website link will be ready shortly after.
